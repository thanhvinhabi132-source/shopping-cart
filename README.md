# Shopping Cart Program in C

Một chương trình nhỏ bằng ngôn ngữ C giúp tính toán tổng tiền khi mua sắm.

## Chức năng
* Nhập tên món đồ (xử lý khoảng trắng bằng `fgets`).
* Nhập giá và số lượng.
* Tính tổng tiền và in ra định dạng đơn vị tiền tệ.
